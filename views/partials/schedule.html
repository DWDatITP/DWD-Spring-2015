Week 1: Introduction and Node.js Install

March 25, 2015

Install fest! This week we will work on installing or otherwise setting up the following:

  • Node.js
  • Git
  • Github account
  • Github App
  • Heroku Toolbelt
  • Heroku Account
  • Select and install a text editor if you don't have one (Sublime Text or Atom recommend)

We'll review basic web concepts and choose projects. We'll review using github and issuing pull requests.

In class:

  • Download and install Node: http://nodejs.org/download/
  • Download and install Git: http://git-scm.com/
  • Create an account (free) at Heroku
  • Install the heroku toolbelt: https://toolbelt.heroku.com/
  • Create an account (free) at Github
  • Download and install the Github App for Mac or Windows.
  • Install "nodemon": `npm install --global nodemon`
  • If you haven't already done so, sign up for the class Email List

Exercises:

  • Nodejs first look: http://www.lynda.com/Nodejs-tutorials/Nodejs-First-Look/101554-2.html - Sections 1, 2, 3
  • Node beginner book: http://www.nodebeginner.org/. The book is not free but there's a lot of free preview content available at the site.
  • Need more help on Javascript basics? Short JavaScript primer.

Examples:

  • ExampleServer - Week 1

Assignments:

  • Decide what your project will be. You can always change it later, but you should make a decision today.
  • When you've chosen an idea, post it to the class email list or email me.
  • Change your app code somehow (it can be a small change) and re-deploy it to heroku.

Example project ideas:

  • Image curation/collection (Pinterest)
  • Bookmark aggregator (Delicious)
  • Blog/CMS (Wordpress)
  • Microblogging Site (Twitter)
  • Arduino Datalogger (API)
  • Social Network
  • Address Book
  • Video Aggregation Site
  • Photo Sharing Site
  • List Making App
  • To-Do App

Optional resources:

  • Preview Heroku/Node tutorial: https://devcenter.heroku.com/articles/getting-started-with-nodejs
  • Getting started with the Github Mac app: http://mac.github.com/help.html
  • Git Essential Training tutorial on Lynda: http://www.lynda.com/Git-tutorials/Git-Essential-Training/100222-2.html
  • In The Beginning Was The Command Line. (Fun read about the command line as the first computer user interface.)
  • Interactive Git Tutorial from Github
  • How to use Git and Github Udacity course
  • JavaScript Basics Udacity Course
  • Intro to HTML and CSS Udacity Course
